## Automotores

CRUD de Automotores para sistema de gestión empresarial.
Permite importar listado de vehículos desde listados en Excel.


##
Patrón de diseño MVC \
Creación del modelo de base de datos en MySQL. \
Migración de tablas y relaciones a través de Eloquent.\
Creación de tablas provincias y localidades mediantes seeds. \
Modelado de clases. \
Creación de rutas
